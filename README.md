# About Me

I'm a **blockchain security researcher** having experience in **public contests** and currently working at **Nethermind**. I have audited a diverse range of decentralized systems across **Solidity, Rust, and Cairo**, including **EVM in Cairo infrastructure (Kakarot)**, **Liquid Staking (LST/DVT)**, **Cross-chain Bridges**, **Appchains**, **Concentrated Liquidity AMMs**, **Restaking Adapters**, **SocialFi**, **AI-integrated GameFi**, **Capital-efficient Stablecoins**, **privacy-preserving identity protocols (ZK/OPRF)**, **Vault infrastructures (ERC4626/UUPS)**, **Incentivized Action Markets**, **distributed off-chain nodes**, **MEV-boost sidecars**, and **automated DCA/yield strategies**.
